# Descrição: 
Aplicação web para localização e filtragem de termos específicos em documentos TXT e PDF.

#Guia de Execução:

1. Clona o projeto do GitHub.
2. Entra na pasta do front.
3. Instala as dependências: npm install
4. Inicia o React: para rodar npm run dev
5. Abre no navegador:http://localhost:5173
